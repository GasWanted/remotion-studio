# Remotion Studio

AI Animation Review Tool — review, pick, and remix AI-generated animation variants for video production.

## What it does

When producing animated explainer videos, AI can generate multiple visual variants for each sentence of narration. This tool provides a human review interface to:

- **See 9 variants** side-by-side in a 3x3 grid
- **Read the narration** that each animation accompanies
- **See the previous scene** to check visual continuity
- **Pick the best variant** with one click
- **Remix with text** — describe what you want changed and get 9 new variants
- **Track progress** across all scenes in a timeline strip

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- **Next.js 15** — App router
- **@remotion/player** — Embed Remotion compositions
- **Tailwind CSS** — Styling
- **TypeScript** — Type safety
